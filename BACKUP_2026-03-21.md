## 🔧 统一配置

### 1. 数据读取方法（5 个市场一致）

**loadFeishuData()**:
```javascript
async loadFeishuData() {
    const response = await fetch('data/trades_{market}.json?t=' + Date.now());
    const data = await response.json();
    const actions = data.trades.map(t => ({
        '交易类型': t.type,
        '市场': data.market,
        '代码': t.code,
        '名称': t.name,
        '成交价': t.price,
        '数量': t.quantity,
        '成交金额': t.amount,
        '交易日期': t.date,
        '状态': t.status,
        '备注': t.reason
    }));
    this.processActions(actions);
}
```

**processActions()**:
- 统一计算方法
- 统一字段命名
- 统一数据源状态显示："本地 JSON 数据 ✓"

## 🌐 信源配置

### 腾讯财经 API
- **A 股/港股**: `https://qt.gtimg.cn/q={codes}`
- **代码格式**: sh/sz + 代码

### Yahoo Finance API
- **Endpoint**: `https://query1.finance.yahoo.com/v8/finance/chart/{symbol}`
- **加股代码**: {CODE}.TO (如 RY.TO, ENB.TO, SHOP.TO)
- **美股代码**: 直接代码 (如 NVDA, MSFT, AMZN)
- **数字货币**: {CODE}-USD (如 BTC-USD, ETH-USD)
