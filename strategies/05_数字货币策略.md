### 数据源

- **方式**: Fetch API
- **刷新频率**: 每 30 秒

### 代码实现
```javascript
async fetchPrices() {
    try {
        // 使用 Gate.io API 获取数字货币实时行情
        const url = 'https://api.gateio.ws/api/v4/spot/tickers?currency=USDT';
        const response = await fetch(url);
        const data = await response.json();
        
        // 创建价格映射
        const priceMap = {};
        data.forEach(ticker => {
            const symbol = ticker.currency_pair.split('_')[0];
            priceMap[symbol] = {
                price: parseFloat(ticker.last),
                changePercent: parseFloat(ticker.change_percentage)
            };
        });
        
        // 更新持仓价格
        this.stocks.forEach(stock => {
            const ticker = priceMap[stock.code];
            if (ticker) {
                stock.price = ticker.price;
                stock.dayChangePercent = ticker.changePercent.toFixed(2);
                stock.profitRate = ((stock.price - stock.cost) / stock.cost * 100).toFixed(2);
            }
        });
        
        this.apiStatus = 'Gate.io ✓';
        this.apiStatusClass = 'success';
        
    } catch (e) {
        console.error('获取数字货币价格失败:', e);
        this.apiStatus = '获取失败 - 请使用 VPN 访问实时行情';
        this.apiStatusClass = 'error';
    }
}
```

## 📝 备注

### 已知问题
1. ⚠️ Gate.io API 在国内可能需要 VPN 访问
2. ⚠️ CORS 限制：部分浏览器可能阻止跨域请求

### 特殊处理
1. **24 小时交易**: 无休市判断，持续刷新
2. **USDT 现金**: 单独显示可用 USDT 余额
3. **演示模式**: API 不可用时显示演示数据

### 备选 API
如果 Gate.io 不可用，可考虑：
1. **币安 API**: `https://api.binance.com/api/v3/ticker/24hr`
2. **CoinGecko**: `https://api.coingecko.com/api/v3/simple/price`
3. **OKX API**: `https://www.okx.com/api/v5/market/tickers`

### 维护建议
1. 定期检查 API 可用性
2. 建仓时更新 POSITIONS 数组
