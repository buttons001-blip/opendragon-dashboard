## 🔌 API 使用汇总

### 腾讯财经 API (主力)
- **适用市场**: A 股、港股、美股
- **地址**: `https://qt.gtimg.cn/q=`
- **特点**: 国内可用、稳定、支持 JSONP
- **前缀规则**:
  - A 股：`sh`/`sz` + 代码
  - 港股：`hk` + 代码
  - 美股：`us` + 代码

### Gate.io API (数字货币)
- **适用市场**: 数字货币
- **地址**: `https://api.gateio.ws/api/v4/spot/tickers`
- **特点**: 支持 CORS、国内可访问
- **备选**: 币安、CoinGecko、OKX

### 本地 JSON (加股)
- **适用市场**: 加股、部分指数
- **文件**: `data/indices.json`, `data/ca_crypto.json`
- **更新**: 后端脚本定时更新
