# 收益率计算规则（2026-03-28 修正）

## ✅ 正确公式

**收益率 = 实际盈亏 / 本金 × 100%**

### 详细说明

| 指标 | 公式 | 说明 |
|------|------|------|
| 实际盈亏 | 持仓盈亏 + 已实现盈亏 | 总盈亏金额 |
| 持仓盈亏 | ∑(当前价 - 成本价) × 数量 | 未实现盈亏 |
| 已实现盈亏 | ∑(卖出价 - 买入价) × 数量 | 已平仓盈亏 |
| 本金 | 初始充值金额 | 投入的总资金 |
| **收益率** | **实际盈亏 / 本金 × 100%** | **正确公式** |

---

## ❌ 错误公式（已废弃）

- ~~收益率 = 实际盈亏 / 已投入 × 100%~~ - 错误！
- ~~收益率 = 总盈亏 / 已投入 × 100%~~ - 错误！

## 💡 为什么用本金而不是已投入？

1. **反映资金利用效率** - 即使只投入部分资金，收益率也应基于总本金
2. **避免误导** - 已投入少时，收益率会虚高
3. **统一标准** - 所有市场使用相同计算方式
4. **符合行业标准** - 券商、基金都使用本金计算
